PROJETO DE AUTOMAÇÃO - ANÁLISE DE TENDÊNCIAS YOUTUBE (STERLING & DRAPER)

Pedro Capoulade
Data: 20/02/2026

1. DESCRIÇÃO DO PROJETO
Este projeto automatiza a análise de vídeos em alta no YouTube para auxiliar os gerentes de planejamento na tomada de decisões diárias. O dashboard consolida dados de múltiplas regiões, permitindo identificar categorias dominantes e comportamentos específicos por país.

2. ARQUIVOS INCLUÍDOS NO ZIP
- trend_by_time.csv: Base de dados utilizada para a construção das análises.
- Apresentacao_YouTube_Trends.pdf: Relatório com os principais insights de negócio e prints do dashboard.
- readme.txt: Instruções e informações sobre o projeto.

3. LINK DO DASHBOARD INTERATIVO (TABLEAU PUBLIC)
Acesse o dashboard completo e funcional através do link abaixo:
https://public.tableau.com/app/profile/pedro.capoulade/viz/YouTubeTrendingAnalysis/YouTubeTrendingAnalysis-ContentPerformanceDashboard

4. INSTRUÇÕES DE USO
- Filtro de Data: Permite analisar períodos específicos de tendências.
- Filtro de Região: Ao selecionar um país, todos os gráficos (Áreas, Pizza e Tabela de Realce) são atualizados automaticamente para refletir os dados locais.
